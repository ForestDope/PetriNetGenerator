A user interacts with a multimedia player. From the main selection screen, they can choose to play a video, play an audio track, or exit the application. If a video is selected, it starts playing. While playing, the user can pause or resume the video, and can also toggle full-screen mode. If the video ends naturally or the user stops it, they return to the selection screen. Similarly, if an audio track is selected, it starts playing. While playing, the user can pause or resume the audio. Additionally, while audio is playing, the user can toggle the display of lyrics. If the audio ends naturally or the user stops it, they return to the selection screen, and any displayed lyrics are automatically hidden.
